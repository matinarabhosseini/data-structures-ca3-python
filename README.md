# Data Structures CA3 - Heaps and Binary Search

This repository contains solutions to three Data Structures course assignment problems implemented in Python.

The assignment focuses on heap-based data structures, sliding window processing, binary search optimization, and efficient algorithm design.

---

## Problems

### 1. Sliding Window Median

Given a sequence of numbers and a window size `k`, compute the median of every sliding window.

Concepts used:

- Heap
- Priority Queue
- Lazy Deletion
- Sliding Window
- Median Maintenance

File:

```text
sliding-window-median.py
```

---

### 2. Battleship Placement

Determine the first shot that makes it impossible to place the required number of ships on a battlefield.

Concepts used:

- Binary Search
- Greedy Counting
- Simulation

File:

```text
battleship-placement.py
```

---

### 3. Minimum Baking Time

Given multiple ovens with different baking speeds, find the minimum time required to bake a target number of breads.

Concepts used:

- Binary Search on Answer
- Search Space Optimization

File:

```text
minimum-baking-time.py
```
